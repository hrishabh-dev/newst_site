# **App Name**: NewsHound

## Core Features:

- Headline Display: Display the latest news headlines and links.
- Date-Based News: Fetch news based on a specific date using the SerpApi API.
- Latest News Fetch: Fetch latest news without filtering by date using the SerpApi API.

## Style Guidelines:

- Use a clean, neutral background (e.g., light gray or off-white).
- Use a dark, contrasting color for text to ensure readability.
- Accent color: Blue (#3498db) for links and interactive elements.
- Use a clear, simple layout with a focus on readability.
- Use simple, recognizable icons for navigation and actions.
